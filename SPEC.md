| 機能 | ゲストユーザー | 認証ユーザー（Google） |
|------|--------------|----------------------|
| ゲームプレイ | ✅ | ✅ |

| スコア履歴 | ✅ | ✅ |
| プロフィール管理 | ❌ | ✅ ニックネーム設定・変更 |

## ゲーム一覧

### ⚡ リアクションテスト

- 画面が緑になったら即座にクリックする
- 緑になるまでの待機時間：1〜5秒のランダム
- フライング（緑になる前にクリック）すると失敗
- **スコア：反応時間（ミリ秒）。低いほど良い**

### 🧠 記憶ゲーム

- 1〜9の数字が順番に光るのを覚えて、同じ順番でクリックする
- 開始は3つの数字から。レベルアップごとに1つ増える
- 間違えるとゲーム終了
- **スコア：正解したシーケンスの長さ × 10点**
- キーボード（1〜9キー）でも操作可能

### 🎨 色判別

- 色の名前（赤・青・緑・黄・紫・黒）が、ある色で表示される
- 「文字の意味」と「文字の色」が一致するかを「はい/いいえ」で答える
- 制限時間：30秒
- 間違えるとゲーム終了（時間切れでも終了）
- **スコア：正解数**

### 🔢 計算ゲーム

- 制限時間60秒以内に計算問題に答え続ける
- 不正解でもゲームは続行（次の問題に進む）
- **スコア：レベル × 10点／正解。レベルはスコアに応じて上がる**

| レベル | 問題の種類 |
|--------|-----------|
| 1〜3 | 加減算（1〜20の数） |
| 4〜6 | 掛け算（1〜12の数） |
| 7以上 | 加減乗算（より大きな数） |

レベルアップ条件：スコアが `現在レベル × 50点` を超えたとき

### 🔍 パターン認識

- 数列の次の数字を4択から選ぶ
- 全10問固定

| 問題番号 | パターンの種類 |
|---------|-------------|
| 1〜3問目 | 等差数列（例：2, 4, 6, 8, ?）|
| 4〜6問目 | 等比数列（例：2, 6, 18, 54, ?）|
| 7〜10問目 | フィボナッチ数列・平方数・交互数列 |

- **スコア：正解数 × 10点（最高100点）**

### ⌨️ タイピング

- 表示された英単語をできるだけ早く正確に入力する
- 制限時間：60秒
- 正しく入力すると次の単語へ
- **スコア：入力した文字数 × 10点**
- 結果画面でWPM（1分間の単語数）と正確率も表示される

## スコア管理

### ゲストユーザー

- `user_scores` テーブルにベストスコアのみ保存（更新方式）
- リアクションは低い方が、その他は高い方がベストスコア

### 認証ユーザー

- `auth_user_scores` テーブルに全プレイ記録を追記保存
- ベストスコアの計算はアプリ側で行う（リアクションは最小値、その他は最大値）

---

## データベース

### `user_scores`（ゲストユーザー用）

| カラム | 型 | 説明 |
|--------|-----|------|
| id | SERIAL | 主キー |
| user_name | TEXT | ユーザー名 |
| game_type | TEXT | ゲーム種別 |
| score | INTEGER | スコア（null = 未プレイ） |
| created_at | TIMESTAMPTZ | 作成日時 |
| updated_at | TIMESTAMPTZ | 更新日時 |

### `auth_user_scores`（認証ユーザー用）

| カラム | 型 | 説明 |
|--------|-----|------|
| id | SERIAL | 主キー |
| user_id | UUID | auth.users への外部キー |
| game_type | TEXT | ゲーム種別 |
| score | INTEGER | スコア |
| created_at | TIMESTAMPTZ | 作成日時 |
| updated_at | TIMESTAMPTZ | 更新日時 |

### `profiles`（認証ユーザーのプロフィール）

| カラム | 型 | 説明 |
|--------|-----|------|
| id | UUID | auth.users と1:1対応（主キー） |
| display_name | TEXT | ニックネーム |
| created_at | TIMESTAMPTZ | 作成日時 |
| updated_at | TIMESTAMPTZ | 更新日時 |

**RLSポリシー：** 全ユーザーのスコアを閲覧可能。自分のデータのみ編集可能。

**game_type の値：** `reaction` / `memory` / `color` / `math` / `pattern` / `typing`

## 認証フロー

1. ヘッダーの「ログイン」ボタンをクリック
2. Google OAuth 2.0（Supabase経由）で認証
3. 初回ログイン時はニックネーム設定画面（`/setup`）へ遷移
4. ニックネームは `/profile` ページで変更可能

---

## 効果音システム

Web Audio API を使用して動的に音を生成する。

| 効果音 | タイミング |
|--------|-----------|
| 開始音 | ゲーム開始時 |
| 正解音 | 正解した時 |
| 不正解音 | 不正解の時 |
| 成功音 | レベルアップ・高スコア達成時 |
| 失敗音 | ゲームオーバー時 |
| クリック音 | ボタン操作時 |

- 音量の調整、および ON/OFF の切り替えが可能
- 設定は `AudioSettings` コンポーネントで管理
